Các bước cơ bản như sau : Cài 2 gói Composer + Laravel dưới -> mở thử mục lên vào file .env ( nếu ko có tạo file này copy nguyên file .envxpamle) sửa hostdb, usename, pass trong file .env giống với db kết nối (xem ví dụ cuối file này) -> Mở xampp tạo bấm tạo db mới bấm tiếp import file applestore.sql để tạo csdl ( các bảng và dữ liệu ) , chạy 2 lệnh cmd (  composer update --ignore-platform-reqs và  composer dump-autoload) để khởi tạo vendor -> chạy  php artisan serve để mở cổng và click vào link trả về đó là link truy cập web

Chú ý lệnh cmd : chạy tại thư mục code đang mở

#Dowload
    - Composer
        - https://getcomposer.org/download/
    - laravel
        - https://laravel.com/docs/9.x/installation#installation-via-composer


#Lệnh cmd
    - Chạy server
        php artisan serve
    - Tạo thư mục :
        composer create-project laravel/laravel your-project-name
    - Tạo file Controller
        php .\artisan make:controller WelcomeController
    - Tạo migration
        php artisan migrate
    -Tạo bảng students
        php artisan make:migration create_students_table
    -Tạo model student
        php artisan make:model Student
    - Reload code change in balade views
        php artisan view:clear
    - Tạo model + Controller
        php artisan make:model Student -a

        chạy heroku cmd
    php /app/artisan migrate --seed (lệnh backup tất cả bảng và dữ liệu seed )
    php artisan db:seed(lệnh backup tất cả bảng và dữ liệu seed khuến khích lệnh này vì chạy theo thứ tự trong file database.seed tránh lỗi insert foreign key )
	để chạy 2 lệnh trên thì phải tạo từ 2 lệnh dưới ( dữ liệu bảng (migrate ) + dữ liệu trong bảng seeder)  t

// tạo dữ liệu từ db sẵn có (Seeder laravel tìm hiểu )
    cài gói     : composer require orangehill/iseed
    chạy lệnh : php artisan iseed management_image_banner,order_detail,order_list,product,product_category,product_images,product_information,user,user_level


// tạ mirgrate từ từ db sẵn có   (Migrate Laravel tìm hiểu )
             cài gói : composer require --dev "kitloong/laravel-migrations-generator"
    ệnh generate : php artisan migrate:generate

#Lỗi và cách fix
    - Uncaught Error: Class "Illuminate\Foundation\Application"
    - failed to open stream: No such file or directory in C:\...\vendor\composer\autoload_real.php
        Chạy :
            composer update --ignore-platform-reqs

            composer dump-autoload
            composer update --no-scripts
            composer update

    -PDOException::("could not find driver") khi chạy cmd php artisan migrate
        Vào nơi cài php.exe -> tìm php.ini
        - tìm tới dòng ;extension=pdo_mysql -> bỏ dấu ; trước -> save
